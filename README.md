# College Complaint Management System

A Django-based web application that allows students to submit, track, and manage complaints online. The system provides a centralized platform for complaint registration, status updates, and resolution, improving communication between students and college administration.

## Features
- Complaint Registration
- Complaint Tracking
- Admin Dashboard
- Status Updates
- Complaint Management

## Technologies Used
- Python
- Django
- SQLite
- HTML
- CSS
- JavaScript
